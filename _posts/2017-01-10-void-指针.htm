什么是void *

void*是指针的一种，在C/C++中很常见，void*是指向一个void的类型的指针,void类型的大小在32位和64位系统下均为1字节，我们可以使用sizeof进行查看。

void *如何使用

void *的使用比较简单，可以和任意的指针就行转换。
例如：

  1. int data = 1;
  2. int *int_ptr = &data;
  3. void *void_ptr = int_ptr;
  4. long *long_ptr =  void_ptr;

什么时候需要void *

void *指针一般是用于传参使用，有些的系统库函数声明为void *类型，在传参的时候将别的指针类型转换为void *类型，在函数内再由void *转换为相应的类型。
由于void * 类型可以与其余指针类型就行无障碍的转换，能够使函数更有通用性。
eg:
我们创建一个新的线程，用于创建线程的函数参数必须声明为void *

  1. #include <stdio.h>
  2. #include <pthread.h>
  4. void* fun(void *data_ptr)
  5. {
  6.     int *int_ptr = data_ptr;
  7.     printf("%d\n", *int_ptr);
  8.     return NULL;
  9. }
  11. int main(int argc, char **argv)
  12. {
  13.     pthread_t tid;
  14.     int data = 11;
  15.     int *int_ptr = &data;
  16.     void *void_ptr = int_ptr;
  17.     pthread_create(&tid, NULL, fun, void_ptr);
  18.     pthread_join(tid, NULL);
  20.     return 0;
  21. }

在创建新线程的时候，我们也可以将long * 赋给void *类型的void_ptr，在新线程中打印一个long类型的值。

指针的步长

指针的步长这个概念是指一个指针进行 + 1 时(或者是自增)，它的数值的变化量，也就是说，它所保存的数值变化量。
我们举一个例子就能很好的理解。

  1. int data = 11;
  2. int *int_ptr = &data;
  3. printf("%p\n", int_ptr);
  4. printf("%p\n", int_ptr + 1);

结果输出如下

  1.  $ 0x7fff572b880c
  2.  $ 0x7fff572b8810

两个数值相差相差 0X4, 大小等于该系统中一个int所占的内存

同理long， float 等类型的指针的步长都是该类型所占的内存大小。

因为指针的步长可能不同，不同类型指针的转换后，如果进行解引用有可能输出奇怪的东西。
例如：

  1. int data = 11;
  2. int *int_ptr = &data;
  3. long *long_ptr = (long*) int_ptr;
  4. printf("%ld\n", *long_ptr);

此时，在我的系统中输出结果为：

  1. $ 66746313351841710091
